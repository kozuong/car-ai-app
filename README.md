# Car AI Project

## Cấu trúc thư mục
- `car-ai-backend/`: Flask API (Python)
- `car_ai_app/`: Flutter app (mobile)

## Hướng dẫn chạy backend
```bash
cd car-ai-backend
python -m venv venv
source venv/bin/activate  # hoặc venv\Scripts\activate trên Windows
pip install -r requirements.txt
python app.py
```

## Hướng dẫn chạy frontend
```bash
cd car_ai_app
flutter pub get
flutter run
```
